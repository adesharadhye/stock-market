# Description

This is a simple Django stock market dashboard I built to track Indian market indices and a custom stock watchlist in one place.

The dashboard shows SENSEX, NIFTY 50, and BANK NIFTY, and it also lets users add stocks like RELIANCE, SBIN, TCS, or direct NSE/BSE symbols such as SBIN.NS and 500325.BO.

The watchlist is saved in the browser, so added stocks stay there after refreshing the page. Prices are fetched from Yahoo Finance public market data and refreshed automatically.

This project is mainly focused on learning and practicing Django views, API endpoints, frontend interaction, stock symbol search, and handling live-style market data in a small web app.

## Short Repository Description

Django stock market dashboard for Indian indices, NSE/BSE stock search, saved watchlist, and auto-refreshing Yahoo Finance quotes.
